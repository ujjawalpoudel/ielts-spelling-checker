About This Project

This tool was created by Ujjawal Poudel for personal IELTS listening spelling practice and refined quickly with help from Codex AI.

Future collaborators are welcome to help improve the experience, dataset quality, accessibility, and study features.

Why learners use this IELTS listening spelling website

Many learners search for help with IELTS listening spelling mistakes, difficult dictation words, confusing spellings, and fast exam-style practice. This website is built to solve exactly that problem in one focused place.

You can practice medium and hard word lists, work through non-repeating batches, pause and resume timed drills, and load your own custom spelling words when needed.
