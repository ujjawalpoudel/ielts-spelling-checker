# English Listening Spelling Practice

An independent listening-spelling practice website built for personal IELTS exam preparation, fast repetition, and focused timed batches.

This project was created by **Ujjawal Poudel** and accelerated with help from **Codex AI** so it could be built in a short period of time for personal practice.

## Important Disclaimer

This is an **unofficial practice project**.

It is **not affiliated with, endorsed by, or sponsored by IELTS, IDP, the British Council, or Cambridge**.

The website is intended for independent spelling and listening preparation only.

## What The Website Includes

- Timed spelling practice
- `Medium`, `Hard`, and `All` difficulty filters
- Non-repeating word batches
- Pause and resume support
- Reset current batch support
- Practice history tracking
- Custom JSON upload and pasted word lists

## Why I Built It

I wanted a cleaner personal practice space for difficult listening words, especially the kinds of words learners often confuse under time pressure. Instead of jumping between scattered resources, this project brings the practice into one focused website.
